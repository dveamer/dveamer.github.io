# Install

## Check the version of Ubuntu

~~~
$ lsb_release -a

result >
  No LSB modules are available.
  Distributor ID:	Ubuntu
  Description:	Ubuntu 14.04.3 LTS
  Release:	14.04
  Codename:	trusty
~~~
